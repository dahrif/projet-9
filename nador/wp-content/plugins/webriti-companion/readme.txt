Enhance Webriti WordPress Themes functionality.

== Description ==

This plugin is the companion plugin for the Webriti themes. It creates the homepage sections like slider section, service section, portfolio section, and testimonial section in the Webriti themes.
Webriti Companion is a plugin build to enhance the functionality of WordPress Theme made by Webriti.
This plugin creates repeater controls in the customizer settings allowing you to create a live site without moving out to customizer screen.
